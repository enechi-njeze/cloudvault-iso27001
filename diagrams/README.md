## Purpose

This folder contains the visual diagrams supporting the ISO 27001:2022 ISMS documentation for CloudVault FHX. ISO 27001 is a management system standard and its documentation benefits enormously from visual representations of the Plan-Do-Check-Act (PDCA) cycle, risk treatment flow, and ISMS process relationships. Senior reviewers including certification auditors and the Authorizing Official review diagrams alongside written documentation.

## Diagram Standards

All diagrams must meet the following standards:

- Legible at standard monitor resolution (1920x1080)
- Exported as PNG at minimum 150 DPI
- Include a title block with system name, diagram title, version, and date
- Use a consistent color scheme: navy blue for primary elements, teal for secondary, gold for callouts
- Include a legend where color coding or shapes carry meaning

## PDCA Cycle Mapping to ISO 27001 Clauses

The Plan-Do-Check-Act cycle is the organizing principle of ISO 27001:

| PDCA Phase | ISO 27001 Clauses | CloudVault FHX Activities |
|---|---|---|
| Plan | 4, 5, 6 | Scope, context, leadership, risk assessment, SoA, objectives |
| Do | 7, 8 | Support, operational controls, Annex A implementation |
| Check | 9 | Performance evaluation, internal audit, management review |
| Act | 10 | Nonconformity, corrective action, continual improvement |
